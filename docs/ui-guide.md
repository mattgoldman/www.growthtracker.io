# UI Guide

##Basic Classes

###.fr
	float; right

###.fl
	float; left

###.cl
	clear

###.of
	overflow; hidden

###.dn
	display; none

###.sm-mar-lt
	5px margin left
	
###.sm-mar-rt
	5px margin right

##Button Styles

###Basic Button

	<div class="bttn">
		<span>Download All</span>
	</div>

###Button With Icon

	<div class="bttn">
		<div class="glyph download-icon">D</div>
		<span>Download All</span>
	</div>
	
###Button With Icon and Loader

	<div class="bttn">
		<div class="glyph download-icon">D</div>
		<span>Download All</span>
		<div class="loader"></div>
	</div>

###Invisible Button
Only appears as a button on hover

	<div class="bttn-invis">
		<div class="glyph download-icon">D</div>
		<span>Download</span>
	</div>

###Button Icon Color Classes
Base color is blue

####.icon-green
	Turns icon green on hover
####.icon-red
	Turns icon red on hover
